# Speed conversion

Convert Speed from Meter Per Seconds to Kilometer Per Hour

## Usage

```js
// Commonjs Import
const { speedConversion} = require("@mskian/speed-conversion");

// or ES6 import
import { speedConversion } from "@mskian/speed-conversion";

console.log(speedConversion(6.685));
// output - 24.1 Km/h
```
